## What This Video Demonstrates

A curated toolkit of 9 free tools for making AI-generated UIs look professional: Open Design (open-source Claude Design with 50+ brand design systems), Referral Styles (2,000 design system markdown files), Impeccable Style (/impeccable teach generates design.md + product.md), a design-engineering GitHub skill, KD Icons (AI vector icon generator), Design Spells (UI pattern inspiration), SVGL (tech company SVG library), Cult UI (ShadCN components + blocks), and Untitled UI (largest open-source React/Tailwind component library).

## Workflow Steps

1. Pick a design system from Referral Styles (2,000 options with markdown documentation)
2. Inject the markdown file into Open Design or paste CSS variables into your project
3. Run `/impeccable teach` to generate `design.md` + `product.md` for the project
4. Use the design-engineering GitHub skill for animation and interaction philosophy guidance
5. Source icons from KD (AI-generated vector icons) or SVGL (tech company logos in SVG)
6. Browse Design Spells for UI pattern inspiration before building complex components
7. Drop in Cult UI or Untitled UI components for modals, onboarding flows, and complex UI needs

## Claude Code Integration Points

The Impeccable skill and design-engineering skill integrate directly with Claude Code as slash commands. Cult UI and Untitled UI components can be pasted into Claude Code prompts as reference implementations. Referral Styles markdown files can be attached to CLAUDE.md as persistent design constraints for a project.

## Key Insight

**Combining a curated design system (Referral Styles) + design vocabulary injection (Impeccable) + ready-made components (Cult UI) eliminates 90% of AI slop from vibe-coded UIs** — the stack enforces professional constraints at every layer.

## Reusable Primitives

- Design system → markdown → inject pattern (pick system, download markdown, paste into agent)
- `/impeccable teach` workflow (generates design.md + product.md from one command)
- KD icon → vectorize pipeline (describe icon → SVG → import to project)
- SVGL logos for social proof sections (copy-paste tech partner logos)
- Cult UI onboarding modal pattern (drop-in for new user flows)
- Referral Styles as persistent CLAUDE.md attachment for project-wide design constraints

## Limitations / Gotchas

- Open Design requires a Claude API key; not usable with Claude.ai free tier
- Impeccable requires skill installation; commands not available without setup
- KD has a free tier with generation limits; production icon sets may exceed limits
- Untitled UI marketing templates require a pro plan; component library is free
- Design systems still require customization to match brand identity
- Referral Styles markdown files vary in quality — some systems are better documented than others
